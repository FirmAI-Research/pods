Baby Names Data Set

The following data comes from a data set of baby names maintained by the Social Security Administration. Each file below contains information about the names and genders of all babies born in the United States in a particular year. All files in this directory follow the naming convention yobYYYY.txt, where YYYY is a four-digit year from 1880 to 2018.

For example, the names of babies born in 1995 can be found in the file https://dlsun.github.io/pods/data/names/yob1995.txt.
